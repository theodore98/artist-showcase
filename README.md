# Setup Instructions

## Important: Environment Variables

Before running the application, you **MUST** create a `.env` file in the project root with the following content:

```env
VITE_LAST_FM_API_KEY={FM_API_KEY}
```

Or use your own API key from [Last.fm API](https://www.last.fm/api/account/create).

## Quick Start

1. Create the `.env` file (see above)
2. Run `npm run dev`
3. Open http://localhost:5173
